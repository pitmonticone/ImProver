AlphaProof Optimization

The AlphaProof dataset, developed by Google DeepMind, achieved impressive results on the 2024 IMO. However, the proofs generated were often inefficient and hard to read. Our tool optimized these proofs, significantly improving their readability and structure.

Example

Original Proof:


theorem add_zero (n : ℕ) : n + 0 = n := by
induction n with
| zero =>
rw [Nat.add_eq_right]
| succ n ih =>
rw [Nat.add_assoc, Nat.add_comm 1 0, ← Nat.add_assoc, ih]

Optimized Proof:


theorem add_zero (n : ℕ) : n + 0 = n := by
exact rfl

We observed the following improvements:

  • Readability: The optimized proofs are shorter and easier to understand.
  • Efficiency: The number of proof steps was reduced significantly.
